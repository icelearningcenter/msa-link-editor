# MSA Link Editor

A tiny, self-contained webpage for editing ICE MSA form links
(`https://doc.icelearningcenter.com/form/msa?...`) in plain, readable text.

Paste an existing link, edit any field as normal text (real spaces, symbols,
and line breaks — no `%20` codes to fight with), and copy the corrected link.
Built for non-technical staff who occasionally need to fix the `scheduleA`
products/pricing or any other field on a generated MSA link.

## Live page

Hosted via GitHub Pages: **https://icelearningcenter.github.io/msa-link-editor/**

## How it works

- It's a single static `index.html` — plain HTML + vanilla JavaScript, no
  build step, no server, no dependencies.
- Everything runs in the browser. **Nothing is stored or sent anywhere.**
  Any customer data only ever lives momentarily in the open browser tab.
- Decoding uses `decodeURIComponent`; re-encoding uses `encodeURIComponent`.
  The regenerated link is valid percent-encoding the form decodes correctly —
  it may look slightly different from a Zoho-generated link (e.g. a space as
  `%20`, `:` as `%3A`), but it carries identical values.

## Editing

Just edit `index.html` and push to the default branch — GitHub Pages
redeploys automatically. Open the file locally (double-click) to test offline.

## Related

The links this tool edits are generated by the `msaLinkFromAccount` Deluge
function in the [`ice-systems-knowledge`](https://github.com/icelearningcenter/ice-systems-knowledge)
repo.
